Temporal network generation

The main problem with face-to-face interaction networks is the lack of big networks to train models and simulate dynamics, to overcome this problem we propose to develop an algorithm for generating temporal networks representing face-to-face interactions. The algorithm should be able to specify the number of individuals types. For instance, suppose to generate a network representing a Hospital, then the user can specify to have 10 Doctors, 3 nurses and 10 patients.
